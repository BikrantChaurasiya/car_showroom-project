# Nova Automobiles - Car Showroom Website

A fully functional static car showroom website featuring:
- Browse cars by brand, type (electric, latest, price drop)
- Detailed car pages with image gallery
- Enquiry forms, test drive booking, showroom visit scheduling
- Loan offers, brochures, dealer locator
- Customer support & service pages

## Deployment
This site is ready to deploy on GitHub Pages. Set the root as the deploy source and the entry point is `index.html`.
